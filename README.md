## 开发经历记录

### 已完成
1. 基础功能-按下开关开始计时，指尖触点闭合结束计时
2. 基础功能-输入回车开始计时，指尖触点闭合结束计时
3. 基础功能-通过大循环保证测试可以重复进行，不需要在每次单次测试完之后重新运行程序。
4. 基础功能-将测试数据保存并设置整体测试参数。
5. 测试稳定性-通过在检测上升沿之后进行高低电位判断，增加系统稳定性，排除因为干扰所触发的上升沿。
6. 基础功能-添加测试选择功能
7. 进阶功能-将控制者发出的指令改为发出声音提示测试者。初步添加开始，成功完成和失败完成三种声音。
